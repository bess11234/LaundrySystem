# Requirements
- Python
- Django
- Psycopg2
- DotEnv
- Tailwind

# Installation
```py
pip install django
pip install psycopg2
pip install python-dotenv
pip install django-tailwind[reload]
py manage.py tailwind install
```

# RUN SERVER
```py
py manage.py tailwind start
```
```py
py manage.py runserver
```
